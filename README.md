# 3D 汉诺塔益智游戏
纯 HTML + CSS + JavaScript 实现的 3D 视觉汉诺塔游戏，适合前端入门学习与课程作业。

## ✨ 功能特性
- 3D 立体视觉效果（纯 CSS 实现）
- 支持 3~8 个圆盘自由调节
- 点击选择 + 点击放置交互
- 自动求解算法（递归实现）
- 移动步数统计
- 游戏规则校验（禁止大盘压小盘）
- 胜利动画与提示
- 响应式布局，支持手机/电脑

## 🎮 游戏玩法
1. 点击柱子上的最上方圆盘进行选中
2. 点击目标柱子完成放置
3. 必须遵守：小圆盘只能放在大圆盘上面
4. 目标：把所有圆盘移到最右侧柱子即通关

## 🛠️ 技术栈
- HTML5：页面结构
- CSS3：3D 变换、渐变、动画
- JavaScript：游戏逻辑、递归算法、DOM 操作
